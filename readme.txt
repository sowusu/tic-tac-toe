Design Decisions

For output messages and prompts, I decided to use a disabled, uneditable text view box. I implemented game choice and starter choice using segmented controls. I maintained a simplistic user interface on a single view app.

For my AI, I used the cornering approach which normally yields multiple
winning plays and increases the chance of winning.
